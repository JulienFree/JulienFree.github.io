Max Biglair

2022

Description:

Dans ce jeu on incarne Max Biglair, un renard agent immobilier. Le jeu consiste à se balader dans le niveau à la recherche de différentes ressources afin de réparer une maison.

Si vous consultez cette page web depuis un ordinateur,
vous pouvez cliquer sur l'image pour jouer au jeux.

Gameplay:

Le jeu se joue au clavier et à la souris, le personnage se déplace uniquement à la verticale en utilisant les touches "A" et "S" ou les flèches directionnelles, la souris sert à casser les ressources ou reconstruire les structures cassées. Quand le personnage marche sur une ressource, celle-ci est collecté et affichée dans l'inventaire. Le joueur peut uniquement réparer une structure quand il a assez de ressources. Une fois la maison construite le joueur peut cliquer sur la porte et terminer le jeu.

Conception:

Durant ma formation d'IMD nous avons eu un workshop d'une semaine sur la conception de jeux vidéo avec le logiciel Unity, durant ce workshop nous avons eu 2 jours consacrer à la théorie et 3 jours pour développer notre jeu à l'aide des base acquises durant les précédents jours. Les illustrations ont été faites avec les logiciels Blender et Photoshop et le code du jeu est en C#.
